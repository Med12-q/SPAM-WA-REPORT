> **SPAM WA REPORT** — A sleek, professional dark platform to report suspicious WhatsApp numbers. Report scammers, spammers, and fake accounts in seconds.

<br/>

---

### ✨ Features

| Feature | Description |
|---------|-------------|
| 🛡️ **Secure Reporting** | All reports encrypted and handled confidentially |
| 📱 **WhatsApp Focused** | Specifically designed for WA spam & scam reports |
| 📊 **Live Statistics** | Real-time dashboard with report counts & analytics |
| 🌐 **Multi-reason** | Spam, Scam, Harassment, Fake Account & more |
| 🌙 **Dark UI** | Premium dark violet interface with glow effects |
| ⚡ **Fast** | Built with React + Vite for blazing performance |

---

### 🚀 Tech Stack

```
Frontend  →  React 19 + TypeScript + Vite
Styling   →  TailwindCSS v4 + Custom Glow Animations
UI        →  shadcn/ui components
Backend   →  Express 5 + PostgreSQL + Drizzle ORM
API       →  OpenAPI spec → auto-generated hooks via Orval
```

---

### 📦 Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Med12-q/SPAM-WA-REPORT)

**Manual setup:**
1. Clone this repo and import in [Vercel](https://vercel.com)
2. Set **Framework Preset** → `Other`
3. The `vercel.json` handles the rest automatically
4. Add `DATABASE_URL` to your Vercel environment variables
